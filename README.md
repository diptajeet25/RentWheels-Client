# RentWheels - Car Rental Website

Live Site: https://bucolic-sprinkles-26594b.netlify.app/
## Project Overview
RentWheels is a modern car rental web application that allows users to browse, book, and manage cars easily. It is built using **React.js**, **Firebase Authentication**, and a custom backend API for managing cars and bookings.

## Features

- **Browse Cars**: Users can view all available cars with details such as category, rent price, location, and images.
- **User Authentication**: Secure sign up, login, and Google OAuth integration using Firebase.
- **Booking Management**: Users can book cars, view their bookings, and unbook when needed.
- **Responsive Design**: Fully responsive and mobile-friendly layout for smooth user experience across devices.
- **Interactive Animations**: Animated  hero banner typewriter effect, and smooth section transitions for better engagement.
